Courses

Fundamentals of Programming

This course is about understanding the basic principles of a programming language.

  • Object-oriented programming (OOP): classes, objects, inheritance, polymorphism, and encapsulation.
  • Basic programming concepts: variables, data types, operators, expressions, control structures, functions, and arrays.
  • Algorithms and data structures: sorting and searching algorithms, stacks, queues, linked lists, trees, and graphs.

Web Development

This course is about understanding the basic principles of web development and how to create webpages.

  • HTML: Hypertext Markup Language, used for creating the structure and content of web pages.
  • CSS: Cascading Style Sheets, used for styling web pages and creating visual design.
  • Front-end frameworks: popular frameworks like React, Angular, and Vue that simplify the process of building web applications.

Project Management

This course is designed to provide students with the skills and knowledge necessary to effectively plan, execute, and manage projects.

  • Project planning: how to create a project plan, including scope, timeline, budget, and resource allocation.1
  • Project scheduling: how to develop and manage a project schedule, including Gantt charts and critical path analysis.
  • Project risk management: how to identify and mitigate risks that could impact project success.
  • Project communication: how to effectively communicate with stakeholders throughout the project lifecycle.
